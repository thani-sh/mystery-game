# Village Roads

A tileset for the peaceful starter village. Everything should have a cozy, slightly overgrown natural feel. It includes:
- Grass variations with and without small flowers or pebbles. Small puddles of water or mud are allowed.
- Dirt roads, corner dirt roads, and intersections with variations. It should be possible to create narrow or wide roads by combining tiles.
- Add some variation to the tiles to make them look more natural. For example, some tiles can have more pebbles, some can have more flowers, some can have patches of dirt, etc.

**CRITICAL REMINDER**:
- The generated tiles MUST be extremely clearly defined as equal-sized squares.
- There MUST be NO GAPS, borders, or any spacing between the tiles.
- Fill in grid lines to make the tiles seamless
