# Village Roads

A tileset for the peaceful starter village. Everything should have a cozy, slightly overgrown natural feel. It includes:
- Grass variations with and without small flowers or pebbles. Small puddles of water or mud are allowed.
- Dirt roads and grass transitions. It should be possible to create wide roads by combining tiles.
- Add some variation to the tiles to make them look more natural, while strictly adhering to a single type of lush green grass and a single type of brown dirt.
- Add road tiles for straight edges, inner corners, and outer corners to allow for creating complex, seamless road shapes.
- Focus strictly on seamless tiles. A tile is useless by itself unless it can be connected on all 4 sides.

**CRITICAL REMINDER**:
- The generated tiles MUST be extremely clearly defined as equal-sized squares.
- There MUST be NO GAPS, borders, or any spacing between the tiles.
- Every single tile edge must be seamless. Each tile MUST be able to tile repeatedly and flawlessly with matching edge types.
